# BOTᵡ 101

**BOTᵡ 101** เป็นแหล่งรวบรวมความรู้พื้นฐานเกี่ยวกับเศรษฐกิจ การเงิน การธนาคารกลาง และหัวข้ออื่น ๆ ที่เกี่ยวข้อง
โดยเป็นแหล่งความรู้ที่เป็น open source เปิดกว้างให้ผู้สนใจเข้ามามีส่วนร่วมในการสร้างและปรับปรุงเนื้อหาได้

ถ้าคุณสนใจมีส่วนร่วม เข้าไปดูรายละเอียดที่ [GitHub](https://github.com/botx/101) ได้เลย
